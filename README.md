ATM Interface
Project Description

The ATM Interface is a console-based Java application The project simulates the basic functionalities of an Automated Teller Machine (ATM) by allowing users to create accounts, securely log in using a PIN, and perform essential banking operations.

The application is designed using Object-Oriented Programming (OOP) principles, providing a modular and maintainable architecture through separate classes for account management, banking operations, and user interaction.

Features
Create a new bank account
Secure login using Account Number and PIN
Deposit money
Withdraw money
Check account balance
View transaction history
Change account PIN
Support for multiple user accounts
Logout and exit functionality

How It Works

The application starts with a main menu where users can create a new account or log in using their account credentials. After successful authentication, users can access various banking services such as deposits, withdrawals, balance inquiry, transaction history, and PIN updates. The system manages multiple user accounts and validates transactions before processing them.
